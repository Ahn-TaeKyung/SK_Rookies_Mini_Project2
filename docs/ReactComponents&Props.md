- [x] 상품 검색 컴포넌트 포함
- [x] 사용자가 로그인 상태인 경우 로그아웃 버튼, 마이페이지/관리자 페이지 링크 표시

- [x] Zustand 스토어에서 인증 상태 및 로그아웃 액션 사용
- [x] 로그인 중 사용자 정보 로딩 시 헤더 숨김
